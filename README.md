## Project

This project is a discussion analyser. Essentially, the first version will be a website, that a person can log into and when a discussion starts click a button to start recoring. Once the discussion is over the person can click again to stop the audio recording and then submit, and an analysis of the audio will begin to give an overview of the discussion. At the most basic level, we want it to have a transcript of the discussion with what things each person said (speaker 1, speaker 2...), and tell the user how long each person spoke. Later on we will expand it to analyse what ideas each person was saying, the sentiment of each person and even when someone interrupted someone else.
